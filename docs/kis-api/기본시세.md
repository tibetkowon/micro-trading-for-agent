# [국내주식] 기본시세 API 상세 명세서

## 주식현재가 시세

- **기능**: 주식 현재가 시세 API입니다. 실시간 시세를 원하신다면 웹소켓 API를 활용하세요.

※ 종목코드 마스터파일 파이썬 정제코드는 한국투자증권 Github 참고 부탁드립니다.
   https://github.com/koreainvestment/open-trading-api/tree/main/stocks_info
- **Method/URL**: `GET` `/uapi/domestic-stock/v1/quotations/inquire-price`
- **TR_ID**: FHKST01010100
